# Getting Started

## Prerequisites

- [Visual Studio Code](https://code.visualstudio.com/download) Or [Visual Studio](https://visualstudio.microsoft.com/)
- [Docker](https://www.docker.com/get-started/)

## Local Start

Run the following commands
```
dotnet tool restore
tada app serve
```

## Dev Container Start


Run the following commands
```
tada app serve
```
